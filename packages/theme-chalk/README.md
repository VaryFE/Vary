# var-theme-chalk
> var component chalk theme.


## Installation
```shell
npm i var-theme-chalk -S
```

## Usage

Use Sass import
```css
@import 'var-theme-chalk';
```

Or Use webpack
```javascript
import 'var-theme-chalk';
```

Or
```html
<link rel="stylesheet" href="path/to/node_modules/var-theme-chalk/lib/index.css">
```

##  Import on demand
```javascript
import 'var-theme-chalk/lib/button.css';

// ...
```
